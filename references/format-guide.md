# Discussion Agenda — Format Guide

This file defines the exact output structure for every agenda. Follow it precisely.

## Output filename

`YYYY-MM - Discussion Agenda.md` (using the current month and year)

## Template

```markdown
# YYYY-MM - Discussion Agenda
Source: [Episode/Film Title](URL) — "[Episode/show name or subtitle]"


## A Subtle Fun Thing:

1. [A creative, indirect callback to episode content. 2-3 sentences explaining the idea and why it connects.]

2. [A different creative callback. 2-3 sentences.]

3. [A different creative callback. 2-3 sentences.]


## Arrival Activity - (Low Stakes Welcome Task)

1. [Name tag + a writing-based thematic task. 1-2 sentences.]

2. [Name tag + a drawing/visual thematic task. 1-2 sentences.]

3. [Name tag + a physical or social/interactive thematic task. 1-2 sentences.]


## Opening

Let's jump in. [One short sentence — max 25 words — that frames the episode's core tension and ends with an open question inviting first reactions.]


## Small Groups - Quick Round (Optional)

*Use this section if you want a warm-up before full group discussion. Have guests pair off with someone they don't know well.*

1. [Question 1 — a "which would you pick" or preference question grounded in episode content]
2. [Question 2 — a question connecting episode content to personal experience]
3. [Question 3 — a question about a tension or debate from the episode]


## Full Group - Themes and Questions

Get the whole group seated. *(If Small Groups was used, open with: "Anyone hear something interesting from their partner they want to share?")*


### [Theme Name 1 — short, evocative]:

[Context sentence grounding the theme in a specific episode detail.]

[Open-ended question 1]

[Open-ended question 2]

[Open-ended question 3]


### [Theme Name 2]:

[Context sentence grounding the theme in a specific episode detail.]

[Open-ended question 1]

[Open-ended question 2]

[Open-ended question 3]

[... repeat for 9 total themes ...]


## Clear Ending

*That's the end of the 'official' discussion agenda portion!*

*What worked here, what didn't?*

Any recommendations for what we should watch, read, or listen to for next time?
```

## Structural rules

1. **Subtle Fun Thing**: Exactly 3 numbered ideas. Each should be a different type of callback (e.g., one food-related, one ambient/sensory, one activity-based).
2. **Arrival Activity**: Exactly 3 numbered options. Each must be a meaningfully different *type* of activity — e.g., one writing-based, one drawing/visual, one physical or social/interactive. All should involve a name tag plus a brief task.
3. **Opening**: Always starts with "Let's jump in." Followed by one short sentence (max 25 words) that frames the episode's core tension and ends with an open question. Do not try to summarize the plot — just name the tension.
4. **Small Groups**: Mark as "(Optional)". Include 3 questions. These should be quicker, more personal, and lower-stakes than the Full Group questions.
5. **Full Group Themes**: Exactly 9 themes. Each theme has a short evocative name, a context sentence, and exactly 3 open-ended questions.
6. **Clear Ending**: Use the exact text shown above every time. Do not modify it.

## Tone rules

- Write as a conversation facilitator, not a professor
- Questions should create genuine disagreement at the table
- Pull in references from philosophy, psychology, science, and culture — but as springboards, not requirements
- Use italics sparingly for emphasis on key reframes
- Use bold only for sub-labels within a theme section if needed (e.g., a named sub-tension)
- The overall voice: smart friend who prepared well, not a study guide
